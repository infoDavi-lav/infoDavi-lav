# Security Policy

## Supported Versions
| Version / Branch | Supported |
| ---------------- | --------- |
| main             | ✅        |
| tagged snapshots (se presenti) | ✅ |
| forks / vecchie snapshot | ❌ |

## Reporting a Vulnerability
IT:
- Invia una segnalazione privata a **info.davi.lav@gmail.com** (non aprire issue pubbliche).
- Includi: descrizione, passi per riprodurre, impatto, eventuale PoC, commit/versione e ambiente.
- Tempi: conferma di ricezione entro 3 giorni lavorativi; aggiornamento o piano di fix entro 14 giorni.
- Disclosure: niente pubblicazione finché non c’è una patch o mitigazione concordata.

EN:
- Send a private report to **info.davi.lav@gmail.com** (please avoid public issues).
- Include: description, reproduction steps, impact, PoC if available, commit/version and environment.
- Timeline: acknowledgment within 3 business days; update or fix plan within 14 days.
- Disclosure: no public disclosure until a patch or mitigation is available (coordinated).

## Out of Scope
IT: attacchi DoS volumetrici, brute force, problemi di servizi terzi, configurazioni custom/non standard, phishing/social engineering.  
EN: volumetric DoS, brute force, third-party service issues, custom/non-standard configs, phishing/social engineering.
